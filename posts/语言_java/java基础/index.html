二、常识

 

<1> java中类名首字母需要大写， 即文件的名称首字母是大写；

<2> java中的两种运算符， (1) 或运算| 与 或或运算 ||, (2) 且运算&和 且且运算, 之间的区别是或运算、且运算一定会计算左右两边的值， 而 或或运算 和 且且运算 可能只计算一边的值

正常情况下，都是使用或或运算，而不使用 或运算， 都是使用 且且运算 ，而不使用 且运算

例如：

public class C2 {
    public static void main(String[] args) {

        System.out.println(false && (10 / 0 > 1)); // false
        System.out.println(true || (10 / 0 > 1));  // true

        System.out.println(true & (10 / 0 > 1));  // error
        System.out.println(false | (10 / 0 > 1)); // error

    } 
}

<3> 运算符的优先级


/*

优先级顺序如下

()
!
* / %
+ -
> >= < <= 
==
!=
& && | ||
=

*/

思考:

为什么赋值运算符优先级最低？ 因为只有有了计算结果后才是赋值操作，所以计算运算符的优先级都高于赋值运算

为什么布尔运算符的优先级低于比较运算符？ 因为布尔运算符的两边都必须是布尔值，而布尔值来源于比较，所以比较运算符先执行，得到布尔值，才能使用布尔运算符

为什么比较运算符的优先级低于算术运算符？ 因为比较运算符比较的两边是数值，而算术运算符可以得到数值，所以先通过算术运算符得到数值，才能进行比较运算符
